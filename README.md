# ГП-5. Модели глубокого обучения для анализа объявлений о недвижимости

## Описание проекта

Работа посвящена разработке прототипа интеллектуального помощника для платформы объявлений о недвижимости. Идея проекта — помочь пользователю оценить качество объявления: проверить адекватность цены квартиры и проанализировать текстовое описание.

Проект состоит из двух связанных частей:

1. **Задание 1** — модель для табличных данных: прогнозирование стоимости квартиры на основе характеристик объекта.
2. **Задание 2** — модель для текстовых данных: анализ текстового описания объявления.

Такое объединение задач позволяет рассматривать объявление не только как набор числовых параметров, но и как полноценную карточку объекта, где важны цена, локация, характеристики жилья и качество описания.

## Файлы проекта

* `ГП_5_задание_1_ipynb_.ipynb` — первая часть проекта, работа с табличными данными.
* `ГП5_задание2_(2) (2) (2).ipynb` — вторая часть проекта, работа с текстовыми данными.
* `data.csv` — исходный набор данных для первой части.
* `README.md` — описание проекта и инструкция по запуску.

## Задание 1: табличные данные

В первой части решается задача прогнозирования цены квартиры. В качестве целевой переменной используется цена за квадратный метр после логарифмирования, так как исходное распределение цены имеет сильную асимметрию и выбросы.

В работе выполнены:

* первичная очистка данных;
* анализ полной цены, цены за квадратный метр и сглаженной цены за квадратный метр;
* сравнение новостроек и вторичного жилья;
* анализ влияния комнатности, округа, метро, удалённости от центра и характеристик квартиры;
* обработка числовых и категориальных признаков;
* обучение нескольких нейросетевых моделей;
* сравнение качества моделей по метрикам;
* сохранение экспериментов и артефактов.

Особое внимание уделено первичной аналитике: она показывает, что цена зависит не от одного признака, а от сочетания факторов — типа жилья, локации, комнатности, состояния квартиры и близости к метро. Поэтому для задачи подходят нелинейные нейросетевые модели.

## Задание 2: текстовые данные

Во второй части рассматривается текстовая составляющая объявления. Эта часть дополняет первую: если табличная модель оценивает цену объекта, то текстовая модель помогает анализировать описание квартиры.

В работе используются методы обработки текста и нейросетевые модели для текстовых данных. Это позволяет связать качество описания объявления с общей бизнес-идеей проекта.
